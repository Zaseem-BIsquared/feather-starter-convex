**Step 1: Make route files thin wrappers**

Each route file should be reduced to just:
- The `createFileRoute` call
- Import of the page component from the feature folder
- The `beforeLoad` metadata (title, headerTitle, headerDescription)

Example pattern:
```typescript
import { createFileRoute } from "@tanstack/react-router";
import { DashboardPage } from "@/features/dashboard";

export const Route = createFileRoute("/_app/_auth/dashboard/_layout/")({
  component: DashboardPage,
  beforeLoad: () => ({
    title: "Dashboard",
    headerTitle: "Dashboard",
    headerDescription: "Overview of your account.",
  }),
});
```

Apply this pattern to all 5 page route files. For `-ui.navigation.tsx`, this file may be imported by the layout -- update the layout to import from `@/features/dashboard` instead if applicable. If `-ui.navigation.tsx` is used as a TanStack Router convention file (the `-ui.` prefix), keep it but make it thin (just re-export from feature folder).
